Banking Analytics Project

A complete Banking Analytics project developed using Excel, SQL, Tableau, and Power BI to analyze Bank Loan Management and Credit & Debit Card Operations.
This project transforms raw banking data into meaningful insights that help improve financial decision-making, customer analysis, loan tracking, and transaction monitoring.

📌 Project Overview

This project is divided into two major modules:

1. Bank Loan Analytics

Analyzes loan applications, funded amounts, repayments, interest earnings, and customer distribution.

2. Credit & Debit Card Analytics

Tracks transaction behavior, balances, debit & credit trends, branch performance, and payment methods.

The project focuses on:

Financial performance analysis
KPI tracking
Customer behavior analysis
Regional and demographic insights
Interactive dashboards and reporting

📈 Key Insights
📍 Loan Analytics Insights
Loan disbursement is concentrated in a few states.
Female customers contribute significantly higher loan amounts.
Loan demand peaked during specific months like March and September.
Economic slowdowns impacted loan trends in FY 2020.
💳 Credit & Debit Insights
Transaction methods are almost equally used.
Customer distribution across banks is balanced.
Branch performance remains stable across regions.
